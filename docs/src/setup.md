# Setup

## F-Droid

Install [F-Droid](https://f-droid.org/).

## Termux

Install [Termux](https://f-droid.org/en/packages/com.termux/) and [Termux:Styling](https://f-droid.org/en/packages/com.termux.styling/).

## App Pinning

Enable `App Pinning`.

## Timeout

Set `Screen timeout` to `2 minutes`.
